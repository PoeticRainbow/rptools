# RPTools
This is a very WIP program and is not intuitive or easy to use yet.

# Install
Unzip the release archive from the actions tab into a folder of your choice and add that folder to your PATH environment variable. Then you will just be able to do `rptools ...` in your terminal to use the program.

- split
- stitch
- overlay
- mask
- duplicate
- replace
- atlas
- format
